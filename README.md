# CharacterSkill Website

캐릭터 스킬(Character Skills)에 관한 웹사이트로, 불확실한 미래에서의 확실한 경쟁력에 대한 정보를 제공합니다.

## 기능

- 캐릭터 스킬의 정의와 중요성 설명
- 8가지 핵심 캐릭터 스킬 소개
- 캐릭터 스킬이 중요한 이유
- 캐릭터 스킬을 개발하는 방법
- 문의 양식

## 기술 스택

- HTML5
- CSS3 
- JavaScript (ES6+)
